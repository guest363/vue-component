# suggestionDaData

Выводит 2 input в которых вводимые данные дополняются подсказками от сервиса DaData.
До 1000 подсказок в день бесплатно. После этого при отсутствии денег на счету просто 
перестают всплывать подсказки.

На ввод подсказок установленна ограничение на частоту обращений к серверам дадаты 
не чаще чем раз в 1 секунду.

Идея такая, что пользователь вводит сначала город а уже затем становится активно поле 
адреса. При этом адрес ищется в выбранном городе.

Достаточным для инициализации является параметр :keyForDaData в котором должен быть указан токен 
от сервиса DaData.

Возвращает компонет обьект вида 
{
  city: "",
  address: "",
  fias: ""
}
и для считывания необходимо подписаться на сабытие suggestion, например так v-on:suggestion="editClient($event)"

## Входные параметры по умолчанию

```
inputs: {
 city: {
  type: "city",
  placeholder: "Город, населенный пункт",
  disabled: false
 },
 address: {
  type: "address",
  placeholder: "Домашний адрес",
  disabled: true
},
},
keyForDaData: ''

```
## Выходные параметры

```
{
  city: "",
  address: "",
  fias: ""
}
```

## Установка

Достаточно скопировать файл с расширением "vue" и папку "support"
